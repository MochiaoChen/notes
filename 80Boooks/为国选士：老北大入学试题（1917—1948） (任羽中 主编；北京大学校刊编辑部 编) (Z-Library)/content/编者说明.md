# **编****者****说****明**

1．本书以年份为序，收录1917—1948年间北京大学入学试题。各年份招考情形不同，试题之招生面向、科目分类、考试批次亦不相同，今见文献或详或略呈现以上信息。为显示各层次、各学科试题所考察的知识范围及知识结构，本书依学制、学科、学部、班次、招考批次等分别收录各份试题，不予合并或拆分，由此确定每份试题名称。

2．民国时期用字、用语不尽符合今日之习惯与规范，本书不强为统一。本书依《通用规范汉字表》使用简体字。民国时期标点符号及其使用规则与今日不同，本书以照录原文文献标点为主，个别依《标点符号用法》加以修改。

3．因翻译来源、知识体系规范等原因，试题中有碍理解的科学名词、专有名词等，本书或径改为今用规范字，如将标示化学元素的“轻”“养”“绿”改为“氢”“氧”“氯”等；或括注今用通称于正文之中，如“苛性曹达（注：氢氧化钠）”“氧化第二铁（注：氧化铁）”“绿精（注：叶绿素）”等。

民国时期地名、人名之译名常与今日通用译名不同，本书皆存其原文。对于差异较大以致不易理解者，则将今用译名括注于正文之中，如“胪滨（注：满洲里）”“非麦（注：阜姆）”等。

4．原试题版式、标序不一，本书重为排版，将“右列”“左列”等径改为“上列”“下列”等，并统一试题序号标注格式。

5．为了便于读者了解试题的背景，选择了试题同时期校长、教师、学生等人的文字作为附录，生动呈现其时北京大学的学习生活场景风气。
