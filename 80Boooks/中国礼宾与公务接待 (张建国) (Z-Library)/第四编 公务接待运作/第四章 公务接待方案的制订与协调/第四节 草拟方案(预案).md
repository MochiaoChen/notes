# 第三节 组织协调

召开协调会，对接待任务进行工作分工。一般接待任务，由接待办牵头组织协调会，召集相关职能部门参与，按工作职责进行分工和提出要求，分头实施；重要接待任务，如接待党和国家领导人、省部级领导同志、地方党政考察团等重要公务活动，由于工作指导、工作考察处于非常重要的位置，召开接待工作协调会时，一般都由党委或政府秘书长(相关机关秘书长)，或党委、政府负责接待工作的副秘书长出面进行协调，代表党委和政府召集参与接待工作的有关各方明确任务，具体分工，提出要求，分头实施，并由接待办负责具体组织落实。为了加强对重要、重大或大型接待任务的组织领导，往往需要组织专门工作班子，设立总牵头人、总负责人、总接待、总保卫并对参加接待任务活动的工作人员进行备案。
