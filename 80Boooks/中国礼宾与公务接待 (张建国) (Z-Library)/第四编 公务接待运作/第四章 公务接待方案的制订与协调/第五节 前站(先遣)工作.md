# 第四节 草拟方案(预案)

各级接待办公室根据任务批件的要求，按照领导同志和来宾公务活动内容，由接待办会同任务相关单位编制接待方案(预案)。接待方案(预案)内容包括考察项目、线路设计、之间距离、所需时间、迎送、陪同、会见、座谈、陪餐、会议、食宿、合影、乘车、医疗保健、食品卫生、新闻报道、安全警卫、职能部门、相关单位、日程安排、天气预报、联系电话、市情、考察点简介等。重要接待任务要有2~3套预案，以供领导决策参考。草拟方案(预案)时，具体要做好以下工作。

(1)草拟参与迎送、陪同、会见、座谈、陪餐领导及有关部门负责人活动安排。

在草拟方案时，提出参与和出席的迎送、陪同、陪餐、汇报(座谈)会等人员安排。需要当地领导同志参与接待活动的，由接待办商请对口办公厅协调。

对上级机关领导同志的迎送、陪同规格中央有明确规定，国内同级机关来宾的迎送、陪同规格各地也都有规定，做法基本相同。迎送、陪同规格的确定主要依据来访者的身份和来访目的，适当考虑两地关系。主要迎送、陪同人员的职务级别通常要与来宾身份对等(或相当)、工作对口。

在这里，需要明确的是“对等”并非时时处处都要求双方对等，而只是在一些必需场合双方职务对等就行了。如迎送、陪同人员的职务往往低于来宾职务，但在陪餐、会见、会谈、座谈、签约时一般都会按照对等的原则进行安排。但有时实在没有对口、对等领导，也可以采取变通措施，由相当职级的领导出面。总之，由于各种原因主客身份不能完全对等时，可灵活变通，但相差不要过大。礼仪可以简化，但不能失礼。

重要的公务活动礼仪一般均要设计礼宾程序，如迎宾程序、送宾程序、会见程序、会谈程序、座谈程序、陪餐程序等，做到礼仪规范，程序严谨。

(2)拟订来宾下榻宾馆，安排食宿。食宿安排是生活安排的主要部分。安排食宿要按中央规定执行，同时考虑任务性质、工作需要、生活习惯和来宾要求，为来宾提供休息、工作和活动便利。

重要接待任务需编制食宿安排一览表(包括单位、姓名、职务、房间号、电话号码、餐厅名称、会议地点、作息时间)。住房安排要相对集中，随行秘书、警卫、医生要与领导同志就近安排，领导同志房间未经批准不得擅入。如要会见或拜访，须与接待办工作人员联系，按程序办理。

(3)草拟客团活动日程安排。日程安排是接待方案的核心内容，也是整个接待任务的主体部分，包括项目考察、情况调研、工作汇报、座谈交流、会见会谈、迎送陪同、出席会议、各种仪式等。日程安排是将这些内容，根据活动的需要，以天为单位，具体到时、分，安排每一天的行程和活动。

(4)拟订陪餐有关事宜。公务接待陪餐具有工作性、礼节性。陪餐又称为餐叙(边用餐边谈工作)，陪餐要严格按规定执行。

(5)拟订接待任务车辆，编制乘车安排表。根据公务出行人员情况，按照轻车简从要求，合理使用车型，尽可能安排集中乘车，严格控制随行车辆及人员。重要任务车队不超过三辆中巴车，一般任务车队不超过两辆中巴车。其他活动应根据情况安排集中乘坐大轿车。

严格按照有关规定使用警车，不得违反规定管控交通。确因安全需要必须安排警卫的，应当按照规定的警卫界限、警卫规格执行，合理安排警力，尽可能缩小警戒范围，不得清场闭馆。

(6)拟订来宾考察项目。重要接待任务或新增加的考察点需要进行实地踩点，进一步落实项目准备、行车路线、停车地点、上下车位、参观路线以及考察活动中举行的汇报会、座谈会、会见、餐叙会、合影等场地设施情况。

(7)拟订会议活动场地。会议活动场地安排主要包括汇报会、座谈会、会见、会谈、签约等会议形式。要根据会议形式、规模布置场地。会场一般要求备音响(含录音)，摆放名签、会议材料、与会人员名单、信纸、铅笔、茶具等。会场不摆鲜花、不备水果、不放烟缸。

(8)资料准备。准备省情、市情简介、视察点、参观点、考察点情况简介等。地级以上市的基本情况资料一般不超过1 500字，县(市、区)基本情况资料不超过1 000字，考察点简介不超过500字。

(9)根据需要，提出执行任务的相关单位，如警卫、交警、新闻、考察点、机场、车站、港口、对口业务单位等。

(10)拟订迎送车辆、人员、时间、地点和线路。

接待方案草拟后，由经办领导核准，按审批权限报有关领导进行逐级审批(重要接待任务工作调研方案制定好以后，须征求上级领导部门的意见，征得同意后实施)。接待方案一经批准，接待办应按正常渠道和程序，立即报送参与接待的有关领导，抄送参与接待工作的新闻单位、公安警卫、食品卫生、医疗保障等相关职能部门和单位执行；协调联系机场、车站、港口等单位，按规定为来宾提供必要礼遇；通知考察点及属地管理部门按要求做好考察服务工作。无特殊情况，一般不得更改接待方案的各项内容。如果改变活动内容，则应按程序办理。

各级接待办公室要根据批准后的接待方案编制实施方案(实施细则)，亦称接待安排。内容包括来宾名单、陪同名单、日程安排、迎送安排、住房安排、用餐安排、乘车安排等。

有关接待单位和部门要按照上一级接待办公室编制的接待方案、日程安排，结合本地区、本单位、本部门情况，制定具体的实施方案，包括考察点、内容简介、行车路线、考察点之间的距离和区间运行时间、合影以及其他有关事项。

每一项接待任务都是一个系统工程，涉及上上下下、方方面面、环环相扣，任何一个环节出了问题都会影响整个接待任务的进行。为此，接待工作要严格执行接待任务责任制，做到把每一个环节落实到人、到事、到单位，确保接待工作零失误。
