# 第四章  
  
公务接待方案的制订与协调

按照要求，党政机关各级部门和单位承担重要、重大接待任务都必须制定详细的接待工作方案；一般的公务接待活动也要制定相应的接待方案，并按规定及程序审批后实施。公务接待方案是公务接待任务活动的重要组成部分，是对公务接待任务的工作部署，是指导接待任务全过程的行动指南，是一种具有公文性质的特殊文体，是公务接待工作中一项极为重要的基础性、系统性和综合性工作。公务接待方案在整个接待任务过程中起着统筹、规划、指导、沟通、协调和具体安排的作用，是确保公务接待任务完成，实施组织、管理、运作的有效工作方式。因此，制定好方案、落实好方案在公务接待任务中具有举足轻重的作用。

制定公务接待方案是来宾视察调研、考察访问、学习交流等公务活动的工作要求，需要“顶层设计”。公务接待方案的内容在一定程度上反映了当地党委政府对此项接待任务活动的思想认识和工作思路，反映了当地主要领导同志对来宾的重视程度和对公务接待规定的执行情况。党委政府主要领导同志既是组织领导者，又是主要参与者，对重要的接待任务方案都会认真审核、严格把关、亲自挂帅、亲自指导，有些特别重要的接待任务，往往是上级有关部门直接组织，地方负责落实。公务接待是一个系统工程，是一项全局性的工作，在接待任务活动中，无论职务高低，实际上都是接待人员，只不过每个人在接待活动中所扮演的角色不同而已。

撰写或起草公务接待方案的人员必须熟悉公务接待的有关规定和程序；熟悉省情、市情；有能力驾驭领导考察调研工作的统筹运作和细节安排；有“身在兵位、胸为帅谋”的担当意识，善于站在全局的角度去思考问题、谋划工作；能及时协调和沟通公务接待活动中可能发生的一些具体事宜；具有相当的文字水平和现代办公设备使用技能。

一套科学的公务接待方案不仅体现出制定者对公务接待方针政策的理解和把握， 而且也反映出当地主要领导同志工作思路、工作方法、政策把握、接待艺术等方面的睿智。制定接待方案的工作过程也是一个极好的提高个人综合素质的过程。
