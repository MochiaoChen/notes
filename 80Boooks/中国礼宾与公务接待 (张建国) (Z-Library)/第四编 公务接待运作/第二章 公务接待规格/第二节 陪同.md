# 第一节 迎送

(1)中央领导同志抵离本地时，相应安排党委或人大、政府、政协对口秘书长到机场、车站、码头和辖区边界迎送，省(区、市)党委书记或省长(主席、市长)和其他省级领导在第一考察点迎接，至最后一个考察点送行，或在驻地迎送。不组织迎送队伍，不悬挂或张贴欢迎标语、横幅，不摆放花草，专机、专列前和考察点一律不得铺设迎宾地毯等，礼仪一切从简。

(2)中央国家机关副部级以上领导同志、外省(自治区、直辖市)副省级以上领导同志到来，参加陪同的当地领导在考察点或驻地迎送。

(3)兄弟城市市委书记、市长、市人大常委会主任、市政协主席及所带领的考察团来访，市委、市政府、市人大常委会、市政协秘书长迎送。兄弟城市副市级领导及所带领的考察团来访，一位相关副秘书长迎送。兄弟城市领导来访，秘书长或副秘书长和工作人员到机场、车站或码头迎送；乘汽车来访时，到入(出)高速路口引领，当地市领导同志在参观点、考察点或驻地迎送。
