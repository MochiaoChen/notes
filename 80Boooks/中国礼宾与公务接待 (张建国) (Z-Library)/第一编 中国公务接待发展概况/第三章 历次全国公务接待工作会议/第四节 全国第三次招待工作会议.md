# 第三节 全国交际处处长会议

1959年3月12日至14日，为保证国庆10周年交际接待任务的顺利完成，经请示国务院同意，在北京饭店召开全国交际处处长会议。出席会议的有各省、市、自治区及中央各有关部门的交际处处长共98人。会前，与会同志参加了中共中央、国务院召开的外事工作会议，听了陈毅副总理和国务院外事办公室廖承志副主任的报告，了解了当前国际国内形势，以及今年外事工作的任务，受到了很大的鼓舞。会上，国管局高登榜局长做了关于保证完成国庆10周年交际接待任务的报告，国务院副秘书长齐燕铭讲了话。国家民委、侨委和武汉、北京、河南、江苏、长春等地的同志在大会上发了言。这次会议的主要收获，一是进一步明确了交际接待工作必须为政治服务、为社会主义建设服务的方向；二是交流了经验，增强了完成今年繁重任务的信心；三是充分强调了贯彻全国一盘棋的精神和勤俭办交际接待工作的作风，为进一步发扬协作和贯彻多快好省的精神创造了条件；四是与会同志一致认为，为更好地完成今年繁重的交际接待任务，必须以做好国庆10周年的交际接待工作为重点，整顿工作，提高业务，加强联系，互相支援，不断地加强经验交流工作，以便更好地为社会主义建设服务。此外，这次会议还反映了许多地方接待部门感到执行收费制度有困难和汽车不足等问题。这次会议亦称为“第二次全国招待工作会议。”[[参考资料|注26]]
