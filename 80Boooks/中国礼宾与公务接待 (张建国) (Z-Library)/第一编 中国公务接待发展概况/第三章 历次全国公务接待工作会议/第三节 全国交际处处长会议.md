# 第二节 全国交际接待工作会议

1958年3月15—25日，全国交际接待工作会议在北京前门饭店召开。出席会议的有各省、自治区、直辖市人民委员会的副秘书长、办公厅主任、机关事务管理局局长、交际处处长、中国国际旅行社分支社经理以及中央各部门的交际接待工作负责人等，共170余人。

会上，陈毅副总理和习仲勋秘书长讲了话，国管局犹凤岐副局长做了《反浪费、反保守，为进一步加强交际接待工作而努力》的报告，中国国际旅行社袁超俊经理做了《关于中国国际旅行社工作的报告》。在大会上发言的还有14个省市的22人。外交部、公安部、外国专家局和北京市有关部门负责人做了专题发言。齐燕铭副秘书长做了总结讲话。

陈毅副总理在讲话中阐述了当前我国政治活动的方针与政策，着重讲了交际接待工作的政治性、交际接待工作人员应当如何做到红与专的问题。习仲勋秘书长在讲话中强调交际接待工作是事务工作，也是政治工作，事务不是事务主义，对外宾的衣、食、住、行，对民主人士的统战工作做好了，本身是事务，效果是政治，做不好就有政治影响。这使会议的思想性、目的性更加明确，给与会人员以很大的教育和鼓舞。

这次会议自始至终贯彻了整改和“双反”(反浪费、反保守)精神。这次会议组织参观了北京饭店、和平宾馆、友谊宾馆和北京饭店的反浪费展览；对各地高级招待所的收费问题和高级宾馆、汽车成本核算及定价问题进行了研究，分别提出了方案；对交际部门在组织机构、规章制度、招待标准等方面存在的一些问题进行了讨论，明确了一些原则，要求各地因地制宜具体解决。关于建立全国交际接待领导体制问题，齐燕铭副秘书长在会上发言时强调，各地方交际接待部门要在当地秘书长统一领导下进行工作，不宜形成上下对口关系，国管局交际处可与各省市交际接待部门建立业务联系。

这次会议是第一次全国招待工作会议，是由国务院召开、国管局承办的。会议筹备工作小组由齐燕铭(国务院)、张永厉(二商部)、岳欣(公安部)、陈忠经(对外文委)、艾楚南(财政部)、殷玉昆(北京市人委)、高登榜(国管局)、袁超俊(中国国际旅行社)、犹风岐(国管局)9人组成，齐燕铭为召集人。3月25日，新华社播发了会议召开的消息。4月15日，国务院向各省、自治区、直辖市人民委员会批转了齐燕铭副秘书长《关于全国交际接待工作会议的报告》及两个附件《反浪费、反保守，为进一步加强交际接待工作而努力》的报告和《关于中国旅行社工作的报告》，并要求各地各部门参照执行。[[参考资料|注25]]
