# 第二节 全国城市接待工作协会

全国城市接待工作协会是由直辖市、计划单列市、省会市(自治区首府市)、特区市等城市(不含县级市)的接待部门自愿组成的社会团体。

全国城市接待工作协会，原名“全国部分城市接待部门联谊会”、“全国城市接待部门联谊会”。由广州、郑州、福州、兰州、杭州五个省会城市接待部门共同发起，于1987年10月在广州成立，并召开了第一次全国部分城市接待部门联谊会，出席会议的共11个会员城市。截至2014年8月，该会共召开会员大会18次(见表4—1)，会员城市发展达到85个(90家会员单位)(见表4—2)。

全国城市接待工作协会的宗旨是：坚持党的基本路线，以邓小平同志建设有中国特色的社会主义理论和“三个代表”重要思想为指导，探索社会主义市场经济条件下接待工作的特点和规律，加强市际的相互联系，进一步发挥接待服务功能，提高接待工作的整体水平，更好地为城市改革开放和经济建设服务。

该协会的基本任务是：

(1)贯彻党和国家关于接待联络工作的各项方针政策。

(2)探索社会主义市场经济条件下接待联络工作的特点和规律，努力促进接待事业的改革和发展。

(3)加强城市接待部门之间的团结、友谊和合作，为促进横向联系创造良好条件。

(4)总结交流经验，促进会员单位之间互相学习、取长补短，不断提高接待工作水平。

(5)组织出版《接待与交际》杂志，培育、弘扬、发展优秀接待交际文化，利用刊物为会员城市传递信息、树立形象、展示风采提供舆论园地。

协会设有理事会。理事会成员每届任期四年，可以连选连任。理事会成员通过民主协商选举产生。理事会设理事长、副理事长、秘书长。理事长、副理事长可以连选连任，但最多不超过两届。

理事会会议原则上每两年召开一次。

协会还设有监事会，监事长(单位)一名。

协会年会实行轮流承办制度。

协会成立以来做了大量的富有成效的工作，对于全国公务接待工作的发展起到了积极的推动作用。

由于种种原因，协会自成立以来始终没有进行社团登记。
