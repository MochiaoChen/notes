# 第一节 中央国家机关接待服务总会

中央国家机关接待服务总会是由中央国家机关宾馆、饭店、招待所、疗养院等接待服务单位及其上级业务主管部门以及有关的社会团体和企事业单位自愿组成的联合组织，是经北京市社会团体行政主管机关批准注册登记的社会团体法人。成立于1994年4月。会址设在北京。该会最高权力机构是会员代表大会，选举产生理事会和监事会;理事会选举产生常务理事及会长、副会长、秘书长。该会下设四个分会。现有会员单位312家，其中直属会员103家，分会会员209家(其中冶金分会20家、邮电分会72家、财政分会63家、建银分会54家)。其业务主管部门为国家机关事务管理局。

中央国家机关接待服务总会的主要职责：

(1)根据党和国家的有关方针政策，积极推动接待服务工作的改革和接待服务事业的发展。

(2)对接待服务工作的有关问题，组织开展调查研究，向有关部门提出意见和建议，发挥参谋作用。

(3)组织接待服务工作的经验和信息交流，促进接待服务部门的合作和联合，发挥整体优势。

(4)促进接待服务部门的社会主义精神文明建设。

(5)维护会员的合法权益，协调与有关部门的关系。

(6)开展接待服务专业业务技术培训，促进服务人员和管理人员业务技术水平的提高。

(7)受业务主管部门委托，承办宾馆、招待所的等级评定工作。

(8)为宾馆、招待所、疗养院等经营单位提供有关市场信息、经营管理业务咨询等服务。

此外，中央国家机关接待服务总会还承办业务主管部门委托的有关工作事项。[[参考资料|注31]]
