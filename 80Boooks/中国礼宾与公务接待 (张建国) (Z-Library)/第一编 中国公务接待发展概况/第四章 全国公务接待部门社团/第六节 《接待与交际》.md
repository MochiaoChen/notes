# 第五节 《接待工作通讯》

《接待工作通讯》的前身是《河南接待工作通讯》，创刊于1986年12月，1989年3月更名为《接待工作通讯》。《河南接待工作通讯》是河南省接待工作年会的会刊，由河南省接待工作年会理事会、郑州市接待办公室联合主办。全国部分城市接待部门联谊会成立后《河南接待工作通讯》更名为《接待工作通讯》，作为全国部分城市接待部门联谊会的会刊，由全国部分城市接待部门联谊会、河南省接待工作年会主办，郑州市接待办公室承办。

《接待工作通讯》是我国改革开放初期的时代产物，是全国公务接待系统第一份行业工作期刊，旨在宣传党和国家关于公务接待工作的方针、政策，交流接待工作经验，传播公务接待改革信息，普及公务接待业务知识，提高接待工作水平和服务质量，为全国公务接待部门服务、为党和政府工作大局服务。它辟有“接待业务”、“经验交流”、“工作研究”、“文件摘编”、“接待动态”、“导游知识”、“旅游名胜”、“饭店管理”、“客房服务”、“餐厅服务”、“烹调技术”、“名城风味”等栏目，专业性、知识性、指导性较强。

《接待工作通讯》为季刊，16开本。从1986年12月至1993年6月，共出版17期。

《接待工作通讯》从1994年3月更名为《接待与交际》，由沈阳市接待办公室承办。
