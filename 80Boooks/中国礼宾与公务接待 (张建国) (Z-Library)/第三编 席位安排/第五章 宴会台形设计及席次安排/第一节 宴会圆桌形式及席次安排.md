# 第五章  
  
宴会台形设计及席次安排

宴会按礼宾规格分为国宴、正式宴会、便宴三种。按宴会形式可以分为中餐宴会、西餐宴会和中西合餐宴会。国宴是以国家名义招待宾客，处处体现国家形象，因而是最高规格的宴请形式，一般地方很少举行。各地举行较多的是正式宴会和便宴。正式宴会又有欢迎宴会、答谢宴会、工作宴会之分。便宴与正式宴会相比显得较为灵活、宽松，规格相对较低，在菜品上往往以地方特色风味为主。

宴会台形设计和席次安排是宴会的重要内容，是指将宴会所用的餐桌按一定要求排列组成的各种格局。

宴会台形设计的总体要求是：突出主宾席，也就是首席(主桌)。主宾席应置于显著的位置，通常主宾席的台面应大于其他各席。

宴会所用桌(台)应排列整齐有序，疏密间隔适当，既方便就餐，又便于服务。

宴会桌次的高低是以离主桌位置远近而定的，右高左低。两桌以上时要摆席次签，即台号(桌号、席号)。有时为照顾基督教国家忌数字“13”的习惯，第13桌往往改为第12A桌。两桌以上的宴会，其他各桌第一主人的位置可以与主桌主人位置同向，也可以，以面对主桌的位置为主位。

席次排列的基本原则：圆形餐厅餐桌居中为上；餐桌横排时以右为上；有主席台时餐桌临台为上。

宴会按规模又可分为：小型宴会(1~10桌)；中型宴会(11~20桌)；大型宴会(21桌以上)。

大、中型宴会由于人多、桌多、应根据宴会的规模，将宴会厅分成主宾席区和来宾席区等若干服务区。

大型宴会主宾席区，一般设五桌，即一主四副。中型宴会主宾席区，一般设三桌，即一主二副。(主宾席副桌的多少不能一味讲究对称，而应该根据上主宾席的人数情况而定。所以，有时就不一定对称，如一主三副或一主五副)。

大型宴会的主宾区与来宾区之间应留有一条较宽的通道，其宽度应大于一般来宾席桌间的距离，如条件许可至少不少于2米，以便宾主出入席间通行便利。

宴会如有乐队伴奏，可将乐队安排在舞台上或主宾席的两侧，也可安排在宴会区的最后一排周围。如果是席间伴有歌舞表演，则席位安排可参照本编“团拜会席位、席次安排”一章。

宴会台形设计注意事项：

第一，主桌应面向餐厅正门，能够纵观宴会厅的位置。

第二，注意对主桌进行装饰，主桌的台布、餐椅、餐具、台面设计等也应与其他餐桌有所区别，但整体应该是和谐一致的。

第三，要有针对性地选择台面。一般直径为150厘米的圆桌，每桌可坐8人；直径为180厘米的圆桌，每桌可坐10人；直径200~220厘米的圆桌，可坐12~14人。总之，主桌应根据人数的多少选择相应规格的圆桌或长台。

第四，台形排列应根据餐厅的形状和大小及宴会人数来安排。桌与桌之间的距离不小于1.5米，餐桌距墙的距离不少于1.2米。
