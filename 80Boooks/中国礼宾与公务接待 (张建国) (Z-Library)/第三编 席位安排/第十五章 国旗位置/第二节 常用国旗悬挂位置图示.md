# 第一节 礼仪要求[[参考资料|注48]]

在悬挂和使用中国国旗时要注意下列礼宾次序或礼仪要求。

(1)升挂国旗应将中国国旗置于显著的位置。在中国国旗与其他旗帜同时升挂时，应将中国国旗置于中心较高或者突出的位置。

(2)在列队行进时，中国国旗应当在其他旗帜之前。

(3)当各国国旗并挂时，各国国旗的面帜应大体相等，高度应一致。升挂时必须先升挂中国国旗，降落时最后降中国国旗。

(4)当一列并排时，以旗面面向观众为准，中国国旗在最右方。

(5)在单行排列时，中国国旗在最前面。

(6)在弧形或从中间往两旁排列时，中国国旗在中心。

(7)在圆形排列时，中国国旗在主席台(或主入口)对面的中心位置。

(8)当中华人民共和国国旗与中国共产党党旗并列时，党旗居左，国旗居右。[[参考资料|注49]]

(9)在中国境内举办双边活动需要悬挂中国和外国国旗时(包括联合国国旗)，凡中方主办的活动，外国国旗置于上首；对方举办的活动，则中国国旗置于上首。

(10)根据《国务院关于在香港特别行政区同时升挂使用国旗区旗的规定》，凡国旗和区旗同时升挂和使用时，应将国旗置于中心、较高或者突出的位置，凡国旗和区旗同时或者并列升挂和使用时，国旗应当大于区旗。国旗在右，区旗在左；在列队举挂国旗和区旗行进时，国旗应当在区旗之前。
