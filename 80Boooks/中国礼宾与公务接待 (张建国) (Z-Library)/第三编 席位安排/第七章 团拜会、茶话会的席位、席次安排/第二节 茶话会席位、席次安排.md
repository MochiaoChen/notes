# 第一节 团拜会席位、席次安排

团拜会是“团体拜年”的意思，是一种集体祝贺节日的礼仪形式，同时也是一种公务活动的会议形式。这种形式既能体现民族风俗，又有益于融洽内部关系，我国党政机关、事业单位、社会团体通常在春节前夕举行团拜会。

团拜会的一般程序是：首先，领导致辞讲话。其次，领导向来宾拜年。领导同志分别到各桌同与会人员互相拜年、互致问候，此时工作人员应乘领导同志离席之际，迅速将会议形式摆放的1号桌席位调整至观看演出的形式(其他席位、席位不变)。再次，观看文艺演出。最后，结束。

团拜会席位、席次安排形式同样适用于联欢晚会、茶话会等伴宴式的餐叙、茶叙活动的席位、席次安排。
