# 第二章  
  
会议主席台席位安排

会议分为设主席台和不设主席台两种形式，设与不设主席台要视会议需要而定。一般来说会议都设有主席台，如党代会、人代会、政协会、庆祝会、表彰会、动员会、记者会、报告会等。而工作汇报会、座谈会、研讨会、讨论会等会议因规模较小，且相互之间需要互动交流就不一定非要设主席台。

主席台的席位安排是根据会议的主题、性质、类别、内容、规模、规则、程序，以及职务、姓氏、单位、场地等多种因素综合进行考量的，形式多样，极具变化。

不同的会议对主席台席位的安排有着不同的要求。有些会议主席台席位是固定的；有些会议主席的席位是相对固定的，并随着会议的进程而不断变化的；有些会议主席台的席位是按照会议程序和规则进行的(如全国人大、政协“两会”)。对此必须予以充分了解，并根据会议需要及时调整席位。为了提高会议的效率和质量，营造良好的会场秩序，使与会人员迅速、准确入席就座，会务工作应越来越趋于规范化，在席位安排上除摆放席位签外，还要绘制会议席位安排一览表，使与会人员一目了然。

会议主席台席位安排的基本规则是：单数以居中为上，然后按先左后右、一左一右的顺序排列(左为上)；双数以右为上，以左为次，然后按先右后左、一右一左的顺序排列(双数以右为上，既符合2号人物始终在1号人物左手位的原则，又符合国际惯例“右”为上的原则，这是对国内公务礼宾活动席位安排的创新发展；单数则与国际惯例相反，仍采取居“左”为上，具有中国特色)；双数按单数排时，主位始终处于正中间，两边呈不对称形状。如果设有报告席(发言席)，则报告席位于主席台的右前方。

根据规模大小、人员多少，主席台的台型可以是一排、两排、三排，或者更多排(甚至分区域设立)。特别是一些大型活动仪式的主席台，主要领导按礼宾次序安排座位，其他嘉宾客人座位则采取只分区而不排座位，或者安排在观众席就座的方式。主席台要讲究台面效果，突出前排、突出主位，做到美观大方和协调对称。一般来说，前排人数少，后排人数多于前排，至少后排不能少于前排人数。

会议主席台的布置，按照务实从简的精神，工作性会议原则上不得铺设红色地毯、摆放花草和绿色植物等。要根据会议需要设置背景板，背景板通常由下列内容组成：会议的徽标，会议的名称、缩写和会议的举办时间和地点，会议的主办方、承办方等办会单位的名称。当会议主席台采用悬挂横幅的传统做法时，横幅一般是红底白字。会议名称如有中英文，则中文在上，英文在下。
