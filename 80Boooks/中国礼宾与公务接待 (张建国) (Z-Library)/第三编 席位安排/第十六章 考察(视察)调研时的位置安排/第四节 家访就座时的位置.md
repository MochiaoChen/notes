# 第三节 考察座谈时的位置

考察调研期间召开的座谈会一般也分两种情况：一种是上下级领导工作性质座谈。其席位按上下级关系安排，即上级领导位于面门一侧(或背靠主席台)，下级负责同志位于背门一侧(或面向主席台)。座位按居中为上，先左后右排列(此种安排也可用于工作汇报)。另一种是亲民式的民间座谈交流，形式相对较为灵活。其席位安排重在突出一线人员(企业、农村、街道社区等基层单位代表)，重在深入群众，使基层群众代表与领导同志有更近距离的接触与交流。在座位安排上主要领导居中，并在主要领导左右安排一线人员(人数视情况而定，通常1~3名为宜)，然后是地陪主要领导左右各一位。其他一线人员按顺序依次就座。参加考察调研活动的有关领导的座位可安排在边位，旁听情况。
