# 第十六章  
  
考察(视察)调研时的位置安排

考察调研是公务活动中各级领导同志一项经常性的工作。考察调研时的站位(席位)安排既是工作要求，又是礼仪要求。由于考察调研活动内容丰富、形式多样，不同的活动主题有着不同的站位(席位)要求，而且有些活动往往具有即时性和群体性，所以参加公务活动考察调研的领导同志，包括陪同领导、考察调研对象的集体站位(席位)安排有较高的礼仪要求。考察调研时的集体站位(席位)安排要把握好以下几个环节：一是主要领导始终居于中心位置(或最佳位置)，二是陪同的1、2号领导分别位于主要领导同志的左手位和右手位，三是突出考察调研对象(突出地陪、突出基层)，四是家访时座位安排注意主客位置(主左客右)。

考察调研时的站位(席位)安排所涉及的礼节主要有四种。
